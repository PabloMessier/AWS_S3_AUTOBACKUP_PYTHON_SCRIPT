Requirements:

Please provide a "credentials.json" wit your AWS credentials in the following format:

{
    "access_key" : "",
    "secret_access_key" : "",
    "BUCKET_NAME" : "include your bucket name here!"
}

Also include a "file_paths.json" this is for the script to be able to interact with your local system.

{
    "source_directories": [
        "C:\",
        "A:\"
    ]
}
